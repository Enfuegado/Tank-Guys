# Tank Guys Multiplayer

Proyecto en Unity con arquitectura de red basada en principios SOLID.

## Features actuales
- Sistema cliente-servidor
- Routing de mensajes tipado (sin strings)
- Arquitectura desacoplada (DIP aplicado)
- Lobby funcional

## Próximos pasos
- Spawn de jugadores
- Sincronización de movimiento
- Predicción de cliente

## Tecnologías
- Unity
- C#
- TCP sockets
